# Torch Studio

A single-file, browser-based **logo studio** — design a wordmark + icon lockup, tune every dimension live, save versions, compare them, and export to SVG/PNG with a settings file you can re-import.

> Built as the design tool behind the **Resonance Labs** mark (the vertical "torch": flame `≋` → signal `=` → breath `~`). It ships with that mark as the default example, but the engine is general — fork it as a base for your own logo.

**Live:** https://torch-studio.vercel.app

## What it does

- **Two-up live preview** — edit logo **A** while a second logo **B** (any saved version, or a live mirror) sits beside it for comparison.
- **Type-anchored geometry** — the icon is *measured from the letterforms*: the flame width snaps to the cap-`R` width and the stroke echoes the font's stem, so the mark always belongs to the typeface.
- **Everything adjustable, in three tabs:**
  - **Shape & spacing** — stroke, wave amplitude, flame/stem/base lengths, vertical gap (edge-to-edge, optionally **locked to the text line-gap**), logo↔text distance, glow.
  - **Type & letterform** — font family (8 Google Fonts incl. variable **Fraunces** with `opsz`/`SOFT`/`WONK` axes), size, line spacing, weight, italic, plus a live letterform metrics study.
  - **Colour** — icons and words coloured **individually** or as a **gradient** (with adjustable angle), a global palette, and surface colours that re-skin the whole page.
- **Versions** — save named snapshots (everything is captured), reload, rename, and **compare** any set side-by-side with a diff table.
- **Undo** — step back through edits, floored at your last save.
- **Export** — SVG (vector, webfont-linked) + PNG (3×, transparent) + a `.settings.json` you can **re-import** to reconstruct the exact logo.

## Run locally

It's one static file. Any static server works:

```bash
npx serve .          # or: python3 -m http.server 8080
```

Then open `index.html`.

## Deploy

```bash
vercel --prod
```

(No build step — it's a single HTML file.)

## Fork it for your own logo

The mark + wordmark are defined inline in `index.html`:
- The wordmark text is the two `<span>`s in `#wm` ("Resonance" / "Labs").
- The icon is generated by `torchGeometry()` / `torchParts()` — swap that for your own SVG path generator and the rest of the studio (tuning, versions, export) keeps working.
